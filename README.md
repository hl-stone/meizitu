# meizitu
妹子图( www.mmjpg.com ) 批量下载 ^_^

## 使用方法

<code>
git clone git@github.com:unclehking/meizitu.git  
</code>
<br />
<code>
cd meizitu  
</code>
<br />
<code>
npm install  
</code>
<br />
<code>
node meizi.js < id > <...>
</code>
